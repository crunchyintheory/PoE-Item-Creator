Path of Exile Item Creator

What is this?

This tool allows anyone to create custom item concepts and modifications to showcase their Path of Exile ideas. It was created in 2018 with a focus on visual accuracy and feature-completeness.The source code is available on GitHub here.

This is a community project maintained by a single developer and is not endorsed by or affiliated with Grinding Gear Games or any official parties. All images and visual elements used on this website are the sole property of Grinding Gear Games.

How do I create items?

In the creator, configure your item's rarity & name, then add any number of lines of text by clicking . This will insert a new affix after the row whose button was clicked. You can insert whatever text you like, it's your item! The affix type choices are there to make it simpler to match the game's formatting.

Adding Images

To add an image to your item, first upload the image to an online image sharing service, then place the URL in the "Image Url" field. The URL should begin with "https://" and end with a file extension (like .png or .jpg).

Exporting Your Items

Your can export your items in two ways: as JSON (for later editing or backup), or as an image (for sharing). Both of these options are accessible by clicking the corresponding buttons at the top right of the item editor.

My foil doesn't work!

Foils do not export properly due to restrictions in the rendering library, html2canvas.js. This issue is not fixable until the requisite changes are made to this library.

My image is blank!

This is likely caused by attempting to load a cross-origin image. This is not a bug with the tool, but rather a security feature of modern web browsers. You can try uploading the image to a different image sharing website that supports cross-origin loading to resolve this.

My image is weird in some other strange and disturbing fashion!

Please open an issue on GitHub containing the image in question and the exported JSON of the item. Also specify your browser version to better narrow down the issue.

Something is Missing / I Found a Bug

Please open an issue on GitHub. If you are dealing with a bug, please try to provide as much information as you can about the steps taken that led to the issue. Please also provide your browser version (and, if it's a display bug, your window resolution).

How long until ___ is added?

Ideally, as soon as the wiki is updated, but I don't always have the time or energy to update this whenever a new item or affix type is added. Sometimes, I also don't know that a certain new thing has been added if I didn't play that league. If you'd like to help, please report what is missing as described above.

Knowing is half the battle, so any reports are greatly appreciated!

Formatting

The following custom formatting specifiers are allowed in all affix value fields: 

%%{{ format[0] }}%%{{ format[1] }}%%{{ format[0] }}%%

You may also use HTML-like formatting (<crucible>My Text</crucible>)

Stash

The stash can be used to store items you have created for later editing. Click the "Save to Stash" button at the top right of the item creator to save the item in your stash, then load the item again by clicking it from the stash.

While you are editing a stashed item, if you try to save it again, it will ask if you would like to overwrite the stashed item you are editing or save it as a new copy. The item currently being edited is indicated by a yellow border on the stash page.
